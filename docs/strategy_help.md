# neural
## Description
    Use neural learning to predict future price. Buy = mean(last 3 real prices) < mean(current & last prediction)
```bash
  options:
    --period=<value>  Period length - longer gets a better average (default: 30m)
    --period_length=<value>  Period length set same as --period (default: 30m)
    --activation_1_type=<value>  Neuron Activation Type: sigmoid, tanh, relu (default: sigmoid)
    --neurons_1=<value>  Neurons in layer 1 (default: 5)
    --activation_2_type=<value>  Neuron Activation Type: sigmoid, tanh, relu (default: sigmoid)
    --neurons_2=<value>  Neurons in layer 2 (default: 5)
    --depth=<value>  Generally the same as min_predict for accuracy (default: 50)
    --min_periods=<value>  Periods to train neural network with from (default: 2000)
    --min_predict=<value>  Periods to predict next number from less than min_periods (default: 50)
    --momentum=<value>  momentum of prediction between 0 and 1 - 0 is stock (default: 0)
    --decay=<value>  decay of prediction, use teeny tiny increments beteween 0 and 1 - stock (default: 0.001)
    --threads=<value>  Number of processing threads you'd like to run (best for sim - Possibly broken (default: 1)
    --learns=<value>  Number of times to 'learn' the neural network with past data (default: 10)
    --learningrate=<value>  The learning rate of the neural network between 0 and 1 - 0.01 is stock (default: 0.01)
```
